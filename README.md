# RepoFit Packet

RepoFit Packet은 프론트엔드 취업 지원 전에 GitHub 프로젝트, 채용공고, 첨부 파일, 보완 메모를 하나의 지원 패킷으로 정리하기 위한 포트폴리오 준비 도구입니다.

기존 RepoFit에서 GitHub 저장소를 불러오고 포트폴리오 후보를 정리하는 기능을 구현한 뒤, 실제 취업 지원 과정에서는 프로젝트 정보, 이력서, 포트폴리오 파일, 채용공고 요구사항, 보완 작업이 흩어지는 문제가 있다고 판단했습니다.

이 프로젝트는 그 흐름을 하나로 묶어, 제출 전 점검 리포트와 이메일 요약을 생성하는 것을 목표로 합니다.

## 핵심 목표

* GitHub 저장소 정보를 불러와 포트폴리오 후보 프로젝트를 확인합니다.
* 채용공고 내용을 입력하고 요구 기술을 정리합니다.
* 지원에 사용할 이력서, 포트폴리오, 캡처 파일을 하나의 패킷으로 묶습니다.
* 프로젝트별 보완 메모와 제출 전 체크 항목을 관리합니다.
* 지원 전 점검 리포트를 생성하고 이메일로 전송합니다.

## 주요 기능 예정

* GitHub 저장소 불러오기
* 제출 후보 프로젝트 선택
* 채용공고 입력
* 프로젝트별 보완 메모 작성
* 첨부 파일 업로드
* 지원 패킷 리포트 미리보기
* 이메일 전송
* 보완 작업 생성 및 체크
